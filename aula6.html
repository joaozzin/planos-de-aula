Aula 6 - Equações e Sistemas de 1º e 2º Graus

Parte 1: Equações de 1º Grau

Introdução às Equações de 1º Grau:
Uma equação de 1º grau é uma igualdade que envolve uma variável elevada à 1ª potência, onde o maior expoente da variável é 1.

Exemplo 1:
x + 5 = 10
Para resolver essa equação, isolamos a variável x:
x = 10 - 5
x = 5

Exemplo 2:
3x - 7 = 8
Para resolver essa equação, isolamos a variável x:
3x = 8 + 7
3x = 15
x = 15 / 3
x = 5

Parte 2: Equações de 2º Grau

Exemplo 1 - Equação do 2º Grau:
Resolva a seguinte equação: x² - 5x + 6 = 0

Solução:
Para resolver esta equação, podemos utilizar a fatoração.

Fatorando a equação:

x² - 5x + 6 = 0

(x - 2)(x - 3) = 0

Agora, resolvemos para "x - 2" igual a zero:

x - 2 = 0

x = 2

E para "x - 3" igual a zero:

x - 3 = 0

x = 3

Portanto, as soluções para a equação são x = 2 e x = 3.

Exemplo 2 - Equação do 2º Grau:
Resolva a seguinte equação: x² - 7x + 10 = 0

Solução:
Podemos resolver esta equação utilizando a soma e o produto das raízes.

Identificamos o coeficiente "a" como 1, o coeficiente "b" como -7 e o coeficiente "c" como 10.

A soma das raízes (S) é dada por: S = -b / a

S = -(-7) / 1 = 7

O produto das raízes (P) é dado por: P = c / a

P = 10 / 1 = 10

Agora, precisamos encontrar duas raízes cuja soma é 7 e o produto é 10. As raízes são 5 e 2.

Portanto, as soluções para a equação são x = 5 e x = 2.

Introdução às Equações de 2º Grau:
As equações de 2º grau são equações algébricas que envolvem uma variável elevada à potência 2 (grau 2). Uma equação de 2º grau possui a forma ax^2 + bx + c = 0, onde a, b e c são coeficientes conhecidos e a ≠ 0.

Fórmula de Bhaskara:
Para encontrar as raízes (soluções) de uma equação de 2º grau, podemos usar a fórmula de Bhaskara, que é dada por:
x = (-b ± √Δ) / 2a

Onde Δ (delta) é o discriminante, dado por Δ = b^2 - 4ac.
- Se Δ > 0, a equação possui duas raízes reais e distintas.
- Se Δ = 0, a equação possui uma única raiz real (raiz dupla).
- Se Δ < 0, a equação não possui raízes reais (raízes imaginárias).

Exemplo 1:
x^2 - 4x + 4 = 0
Para resolver essa equação, podemos utilizar a fórmula de Bhaskara:
x = (-(-4) ± √((-4)^2 - 4*1*4)) / 2*1
x = (4 ± √(16 - 16)) / 2
x = (4 ± √0) / 2
x = (4 ± 0) / 2
x = 4 / 2
x = 2

Exemplo 2:
2x^2 - 5x - 12 = 0
Para resolver essa equação, podemos utilizar a fórmula de Bhaskara:
x = (5 ± √(5^2 - 4*2*(-12))) / 2*2
x = (5 ± √(25 + 96)) / 4
x = (5 ± √121) / 4
x = (5 ± 11) / 4
x = (5 + 11) / 4 ou x = (5 - 11) / 4
x = 16 / 4 ou x = -6 / 4
x = 4 ou x = -1.5

Parte 3: Sistemas de Equações

Introdução aos Sistemas de Equações:
Um sistema de equações é um conjunto de duas ou mais equações que envolvem as mesmas variáveis.

Exemplo 1:
Sistema:
2x + y = 10
3x - y = 4
Para resolver esse sistema, podemos utilizar o método da adição/subtração:
Adicionando as duas equações, eliminamos a variável y:
2x + y + 3x - y = 10 + 4
5x = 14
x = 14 / 5
x = 2.8
Agora, substituindo o valor de x em uma das equações, encontramos o valor de y:
2 * 2.8 + y = 10
5.6 + y = 10
y = 10 - 5.6
y = 4.4
Portanto, a solução do sistema é x ≈ 2.8 e y ≈ 4.4

Exemplo 2:
Sistema:
3x + 2y = 12
x - y = 1
Para resolver esse sistema, podemos utilizar o método da substituição:
Isolando a variável x na segunda equação:
x = y + 1
Agora, substituindo o valor de x na primeira equação:
3(y + 1) + 2y = 12
3y + 3 + 2y = 12
5y + 3 = 12
5y = 12 - 3
5y = 9
y = 9 / 5
y = 1.8
Agora, substituindo o valor de y na segunda equação para encontrar x:
x - 1.8 = 1
x = 1 + 1.8
x = 2.8
Portanto, a solução do sistema é x ≈ 2.8 e y ≈ 1.8

Encerramento

1. Resolva a equação de 1º grau: 3x + 2 = 8

2. Resolva a equação de 2º grau: x^2 - 6x + 9 = 0

3. Resolva o sistema de equações:
2x + y = 7
3x - y = 3
